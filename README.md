A simple GUI that lets you choose a folder in any directory and delete everything in said folder, I plan on adding more buttons so you can choose what and what not to delete. 
This is my first time interacting with the stream in a GUI environment so I wanted to try some new things.
please be careful if you use this and make sure you create a backup of anything you delete if its important.
Again, this will delete EVERYTHING inside a folder, including zips and other folders.
